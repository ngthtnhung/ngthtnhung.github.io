BudTrack — Personal Finance Management App

A comprehensive Android budgeting app for tracking expenses, managing wallets, and visualizing spending with material design and multilingual support.

Overview

BudTrack is a personal finance management application built to help users manage budgets, record expenses, and view financial insights through a clean Android interface.

Features

  • Budget Management: create and manage budgets for expense categories
  • Expense Tracking: record, categorize, and review expenses
  • Analytics: visualize spending patterns and financial insights
  • Wallet Support: manage multiple wallet types including Basic, Credit, and Savings
  • Calendar View: track expenses by date with an interactive calendar
  • Dark Mode: full dark mode support for comfortable viewing
  • Multi-language: Vietnamese and English support
  • Material Design: modern UI with smooth interactions

Technology stack

  • Language: Java
  • Build system: Gradle
  • Design: Material Design 3
  • Testing: JUnit, Espresso
  • Target: Android API level set in build.gradle

Getting started

Run the app from Android Studio or the command line:

./gradlew build
./gradlew assembleDebug
./gradlew assembleRelease
./gradlew test
./gradlew connectedAndroidTest

BudTrack delivers a polished finance app experience with strong budgeting workflows, easy expense tracking, practical analytics, and accessible Vietnamese/English support.
