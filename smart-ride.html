SmartRide — Ride-Sharing Platform

Led business analysis for a ride-sharing platform through requirements, workflows, and interactive UI prototypes.

Overview

This university project focused on business requirements, UX, and product flow for a ride-sharing platform designed to meet user and operational needs.

Work completed

  • Gathered and documented stakeholder requirements for core ride-sharing features.
  • Created user stories, acceptance criteria, and process workflows.
  • Designed UI/UX wireframes in Figma and built interactive prototypes.
  • Validated the platform flow through user journey mapping and scenario testing.

Outcome

The project produced a detailed requirements package and clickable prototypes that aligned business goals with a user-focused ride-sharing experience.
