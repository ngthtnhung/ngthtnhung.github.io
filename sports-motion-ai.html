Project detail

Application of Text2Motion and Motion2Text in Sports Education

University research project using computer vision and motion modeling to deliver golf swing feedback and personalized coaching insights.

Overview

Built an AI coaching system that analyzes sports motion, generates text-based feedback, and evaluates movement quality using pose estimation and diffusion modeling.

Work completed

  • Designed a research pipeline for Motion2Text and Text2Motion use cases.
  • Implemented PoseSwingNet and 3D biomechanics analysis using pose estimation.
  • Developed a feedback system to interpret motion patterns and suggest corrections.
  • Evaluated results with automated performance metrics and model comparisons.

Outcome

The project produced an AI prototype that turns motion capture into actionable coaching feedback, demonstrating the value of data-driven biomechanics analysis for sports training.
