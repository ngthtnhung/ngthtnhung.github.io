NavFlow - Smart Traffic Management System

Designed an indoor navigation solution that improves wayfinding efficiency through user research, workflow design, and usability improvements.

Overview

This project focused on designing a navigation system for indoor spaces with attention to user requirements, path planning, and interface guidance.

Work completed

  • Conducted user requirement analysis and mapped navigation challenges.
  • Created business process designs for indoor wayfinding and support flows.
  • Built UI wireframes and user journey flows to guide navigation tasks.
  • Recommended usability improvements to reduce confusion and improve efficiency.

Outcome

The final concept provided a clearer indoor navigation experience and a stronger foundation for product design in environments like campuses or event spaces.
